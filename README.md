# System Linux Wiki

System is a collection of software additions to the official Arch Linux repositories. It is very simple to use with a monochrome KDE/Plasma 6 desktop and a graphical installer. The installer allows for the preselection of certain groups of software such as desktop environments, programming environments, third-party Arch repos, etc. The default selection includes KDE, C++ Development and the BlackArch repositories.

System is designed for software developers and security professionals. It is not for beginners or Windows/Mac migrants. A working knowledge of Arch Linux is -required-. The current method of install involves creating an ISO file yourself, including building the installer from source. System will eventually distribute more tools than just the BlackArch/Kali/CSI toolsets.
